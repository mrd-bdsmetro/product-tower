# MVP Scope — Solo Founder Edition

> Ship trong 2-4 tuần. 3-5 features. Không hơn.

---

## MVP Checklist

```
□ 1. MUST features chỉ (từ MoSCoW)
□ 2. Max 5 features
□ 3. Ship trong 2-4 tuần
□ 4. 1 core value prop rõ ràng
□ 5. Có thể test với real users
□ 6. Metrics đã define (đo thành công = gì?)
```

## Scope Killer Questions

Trước khi thêm BẤT KỲ feature nào, hỏi:

```
1. "Feature này solve UNMET NEED nào?"
   → Không map = CẮT

2. "Solo founder build trong 1 tuần được không?"
   → Không = phải tách nhỏ hơn hoặc CẮT

3. "Nếu bỏ feature này, MVP còn test được value prop không?"
   → Vẫn test được = CẮT (chuyển sang SHOULD/COULD)

4. "Feature này cần data/users trước không?"
   → CẦN data = đợi. Build infra thu thập data trước.
```
