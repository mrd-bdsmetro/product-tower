# Example: TaskFlow — Feature Scoping (Simple)

## Input
- PMF Score: 47.5% ✅
- HXC: CTO/Tech Leads ở startup 5-10 người
- Core value: Auto status updates from Git

## MoSCoW

| Priority | Feature | Effort |
|----------|---------|--------|
| **MUST** | Git commit → auto status | 5 days |
| **MUST** | Team dashboard | 3 days |
| **MUST** | Slack integration | 2 days |
| **SHOULD** | Client-facing view | 3 days |
| **COULD** | Smart deadline warnings | 3 days |
| **WON'T** | AI task assignment | 15 days |

**MVP: 3 MUST = 10 days**
