# Customer Journey Map — 5 Giai Đoạn

> Vẽ hành trình user từ lúc biết → mua → dùng → trung thành.
> Tìm ra touchpoint đau nhất → đó là CƠ HỘI.

---

## 5 Giai Đoạn

```
 AWARENESS    CONSIDERATION    DECISION      USE         LOYALTY
 "Biết đến"   "Cân nhắc"      "Quyết định"  "Sử dụng"  "Trung thành"
     │              │              │            │            │
     ▼              ▼              ▼            ▼            ▼
  Trigger →     So sánh →      Mua/Ký →    Onboard →    Giới thiệu
  Tìm kiếm     Đọc review     Thanh toán   Dùng daily   Feedback
  Nghe bạn bè  Test/Trial     Setup        Gặp issues   Refer
```

---

## Mapping Tips

1. **1 journey = 1 persona** — không mix personas
2. **Dùng data thật** — interview quotes, analytics, không giả định
3. **Focus vào pain** — tìm 😤 moments, đó là cơ hội
4. **Compare với competitor** — journey của competitor có điểm nào smoother?
