## BÀI HỌC QUAN TRỌNG

### 2-Sided Discovery Challenges

1. **Chicken-egg**: Foodie cần nhiều quán → quán cần nhiều foodie
2. **Different WTP**: Demand = free, Supply = trả tiền
3. **Different interviews**: Phải interview CẢ 2 bên riêng
4. **Journey khác nhau**: Demand journey ≠ Supply journey

### Khi nào DỪNG 2-sided?

```
IF demand WTP = 0 AND supply WTP < $50/tháng:
   → Business model khó survive
   → Cần rethink monetization (ads? affiliate? premium?)
```
