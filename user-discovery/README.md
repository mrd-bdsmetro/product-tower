# 🔍 User Discovery Skill — Product Tower Tầng 4-6

> Build personas. Map needs. Tìm unmet needs. Hiểu user THẬT SỰ cần gì.
> Sai ở đây = build thứ không ai dùng.

---

## 🎯 Skill này làm gì?

Giúp founder/PM chạy **3 tầng khám phá user**:

| Tầng | Tên | Output |
|------|-----|--------|
| **4** | User Personas | 2-4 personas (BAG framework + JTBD) |
| **5** | User Needs (Prioritized) | Ranked needs + RICE scores |
| **6** | Unmet Needs | Cơ hội kinh doanh = cái chưa ai solve |

### Tích hợp best-in-class:
- 👤 **Persona BAG** (Behaviors, Aspirations, Goals)
- 🎯 **Mom Test** (Rob Fitzpatrick) — hỏi đúng cách, không bias
- 📊 **RICE Scoring** — prioritize bằng data
- 🗺️ **Customer Journey Map** — tìm touchpoint đau nhất
- 📐 **Kano Model** — phân loại basic vs excitement
- 🔍 **Opportunity Scoring** — đo mức unmet
