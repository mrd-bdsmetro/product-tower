## 🎯 Skill này làm gì?

Validate xem sản phẩm đã đạt PMF chưa bằng 3 methods:

| Method | Khi nào dùng | Output |
|--------|-------------|--------|
| **Sean Ellis Test** | ≥ 30 active users | PMF Score (40% rule) |
| **Superhuman Engine** | Score < 40%, cần improve | Improvement roadmap |
| **Metrics Dashboard** | < 30 users, pre-survey | PMF signal strength |

### Tích hợp best-in-class:
- 📊 **Sean Ellis 40% Test** — gold standard PMF measurement
- 🚀 **Superhuman PMF Engine** (Rahul Vohra) — iterative improvement
- 📈 **PMF Metrics Dashboard** — 15 leading/lagging indicators
- 🎯 **Value Proposition Canvas** — check fit before survey
