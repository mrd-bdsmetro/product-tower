## Survey (gửi cho ACTIVE users, đã dùng ≥ 2 lần)

### Câu 1 (CORE — PMF Score)

```
Bạn sẽ cảm thấy thế nào nếu KHÔNG CÒN được dùng [sản phẩm]?

☐ Rất thất vọng
☐ Hơi thất vọng
☐ Không thất vọng (không hữu ích lắm)
☐ Tôi không dùng nữa rồi
```

### Câu 2 (HXC Identification)

```
Bạn sẽ giới thiệu [sản phẩm] cho ai? Mô tả ngắn.

[________________________________]
```
> Mục đích: Tìm hiểu user coi xem ai là target user LÝ TƯỞNG 
> trong mắt họ → đó là HXC profile

### Câu 3 (Value Discovery)

```
Lợi ích CHÍNH mà bạn nhận được từ [sản phẩm] là gì?

[________________________________]
```
> Mục đích: Hiểu VALUE PROP qua lời user → dùng cho marketing

### Câu 4 (Improvement)

```
Bạn muốn CẢI THIỆN gì ở [sản phẩm]?

[________________________________]
```
> Mục đích: Tìm gaps → roadmap improvement

## Cách Phân Tích Kết Quả

### Bước 1: Tính PMF Score

```
PMF Score = (Số "Rất thất vọng" ÷ Tổng responses) × 100

Ví dụ: 15 "Rất thất vọng" ÷ 40 tổng = 37.5%
```

### Bước 2: Interpret

```
≥ 40%  → ✅ PMF! Ready to scale
        → Action: Double down, scale marketing

30-39% → 🟡 Almost there
        → Action: Chạy Superhuman Engine → improve

20-29% → ⚠️ Early signals, cần iterate nhiều
        → Action: Focus HXC, fix top gaps

< 20%  → ❌ Chưa có PMF
        → Action: Quay lại T4-6, re-validate needs
```

### Bước 3: Segment HXC

Filter responses "Rất thất vọng" → tìm pattern:
- Họ là AI? (role, level, industry)
- Họ dùng feature nào NHIỀU NHẤT?
- Keyword nào lặp lại trong câu 2 + 3?

→ HXC Profile = "Những người [mô tả] mà dùng [feature] cho [purpose]"

### Bước 4: Gap Analysis (từ câu 4)

```

│ ________________     │ ___    │ [Có/Không]   │

Priority: HXC requests > non-HXC requests
```
